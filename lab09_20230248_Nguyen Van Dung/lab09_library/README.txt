LAB09 - QUẢN LÝ THƯ VIỆN (PDO + MVC + AJAX)

1. CÀI ĐẶT
- Giải nén vào: xampp/htdocs/lab09_library
- Import database/it3220_library.sql
- Start Apache + MySQL

2. LINK CHẠY
Books:
http://localhost/lab09_library/public/index.php?c=book

Members:
http://localhost/lab09_library/public/index.php?c=member

Borrows:
http://localhost/lab09_library/public/index.php?c=borrow

3. CHỨC NĂNG
- CRUD Books (Ajax)
- CRUD Members (Ajax)
- Borrow / Return (nâng cao)

- MVC rõ ràng
- JSON response

4. VALIDATE
- Email đúng định dạng
- Quantity >= 0
- ID tồn tại khi update/delete
